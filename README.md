# Aplikacja kontroli budżetu domowego
Program interfejsowy służący do prostych wyliczeń budżetu domowego i samodzielnej prognozy wpływów i wydatków.
# Autorzy i licencja

Autorzy: Grzegorz Sadłakowski, Patrycja Kurzątkowska

Licencja: MIT
# Funkcjonalności

-Kategoryzacja wydatków

-Wprowadzanie przychodów i wydatków 

-Liczenie salda

-Tworzenie wykresu porównawczego wydatków i wpływów

# Architektura

Język programowania: Python 3.9.13

IDE: Visual Studio Code

Biblioteki: tkinter, matplotlib
